Rewrites your preview links from the address set in the HOME option to the one set in SITEURL.

== Description ==

Can be useful when your WordPress administrative dashboard is hosted on another domain/server than the publicly accessible website.

This plugin can be used as a template when you have more specific requirements for rewriting your preview links.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/siteurl-preview-links` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the ‘Plugins’ screen in WordPress.
